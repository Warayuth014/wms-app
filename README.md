# WMS App

Warehouse Management System — Flutter Mobile App

## Tech Stack
- Flutter 3
- sqflite (Offline Support)
- connectivity_plus

## Features
- Flow 1: รับสินค้าเข้า / รับสินค้าคืน
- Flow 2: Unload / Load Basket
- Offline Support + Auto Sync
- Supervisor Cancel Approval

## Setup
1. Clone repo
2. flutter pub get
3. Update API URL in api_service.dart
4. flutter run
